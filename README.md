# Traffic-Intersection-Trajectory-Prediction-Fusing-Transportation-Rules-and-Diverse-Motion-Behaviors

The model bases on pytorch and python3.6.

Using pip install -r requirements.txt to construct the python environment.

The files of dataset include Veh_tracks_meta.csv, road_information.npz, Veh_smoothed_tracks.csv, TrafficLight_8_02_1.csv, and Ped_smoothed_tracks.csv.

The visual tool can be downloaded from https://github.com/SOTIF-AVLab/SinD.

Using generate_graph.py to produce the multi-graph.

Using train.py to train the model.

Using test.py to evaluate the model.
